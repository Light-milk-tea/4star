# 4star Website

一个简洁的Vue3网站，显示"4star"。

## 部署方式

### 方案1：GitHub Pages（推荐，国内访问较好）
1. 将此项目推送到GitHub仓库
2. 在GitHub仓库设置中启用GitHub Pages
3. 访问 `https://[你的用户名].github.io/4star/`

### 方案2：Vercel（需要翻墙）
- 已部署地址：`https://trae4starqv3z.vercel.app`

### 方案3：Netlify
- 支持拖拽部署，国内访问相对较好

## 本地开发
```bash
npm install
npm run dev
```

## 构建
```bash
npm run build
```
